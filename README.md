# bitcoin-buys

A bitcoin dca bot, that buys bitcoin with eur at regular intervals.
Pretty crude atm, but does the job well enough for me.

## Installation

```
npm run install
```

## Configuring

Create a .env file in the root folder of the project with the variables:
- KRAKEN_API_KEY
- KRAKEN_API_SECRET

## Running

```
npm run start
```
